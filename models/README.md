# Trained Models Directory

This folder should contain the trained machine learning models generated by the training scripts.

## Required Files

Place the following files here after running the training pipeline:

- `model_lr.pkl` - Logistic Regression model
- `model_rf.pkl` - Random Forest model  
- `model_xgb.pkl` - XGBoost model
- `model_nn.keras` - Neural Network model
- `scaler.pkl` - Feature scaler
- `feature_names.pkl` - Feature names list
- `xgb_threshold.pkl` - XGBoost threshold
- `nn_threshold.pkl` - Neural Network threshold
- `ensemble_weights.pkl` - Ensemble model weights
- `ensemble_threshold.pkl` - Ensemble threshold
- `full_comparison.csv` - Model performance comparison

## Instructions

Run the training scripts in order:
1. `python training/01_data_preprocessing.py`
2. `python training/02_ml_models.py`
3. `python training/03_neural_network.py`
4. `python training/04_ensemble.py`

Then copy the generated files from `output/` to this `models/` folder.
